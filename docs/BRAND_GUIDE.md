# LionexAI Brand & Presentation Guide

Colors and typography aligned with the **LionexAI logo** and the platform design system (`frontend/src/app/globals.css`).

## Logo palette

| Role | Hex | Usage |
|------|-----|--------|
| Metallic gold | `#CFA43B` | “LIONEX”, headlines, primary metrics |
| Bright gold | `#E9C566` | Highlights, icons, chart accent |
| Emerald / teal | `#0FA89A` | “AI”, innovation, positive secondary accent |
| Bright teal | `#1ED6C0` | Links, callouts, focus rings |
| Deep black | `#070809` | Slide / app background |
| Charcoal card | `#101216` | Content panels, metric cards |
| Panel gray | `#161A20` | Tables, secondary sections |

**Brand gradient:** `#CFA43B` → `#0FA89A` at **135°** (gold top-left to teal bottom-right).

## Text & status

| Role | Hex |
|------|-----|
| Primary text | `#F6F8FB` |
| Secondary text | `#C4CCD8` |
| Muted text | `#8A94A6` |
| Institutional blue | `#4A92FF` |
| Success | `#1ED6A6` |
| Warning | `#F5B23B` |
| Danger | `#FF4D67` |

## Typography

| Use | Font |
|-----|------|
| Display / titles | **Sora** (or elegant serif for logo-style slides) |
| Body | **Inter** |
| Metrics / code | **JetBrains Mono** |

## Presentation themes

### Dark institutional (recommended)

- Background: `#070809`
- Title: `#F6F8FB`
- Accent 1: `#CFA43B`
- Accent 2: `#0FA89A`
- Cards: `#101216` with border `rgba(255,255,255,0.10)`

### Light variant

- Background: `#F6F8FB`
- Text: `#101216`
- Accents: keep gold `#CFA43B` and teal `#0FA89A`

## Chart colors

1. Primary series: gold `#CFA43B`
2. Secondary series: teal `#0FA89A`
3. Validated / backtest data: blue `#4A92FF`
4. Demo / operational (admin-only slides): red `#FF4D67`

Place the logo on **black or `#070809`** — avoid busy backgrounds behind the metallic mark.
